About

LightAlgorithm is a web application which will help user to visualize Data Structures with the help of computer technologies. This may help many of to simplify their concept of Data Structures with the help of Graphical Representation. As it's name defines, We tried to keep the GUI in dark or neon themed which will be attractable to this new generation. We will try to improve this project in the future and implement more things. The overall process helped us to improve our knowledge and we are sure that it will improve yours also! Thanks!
